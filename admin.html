Email Delivery Setup (Optional)

Messages are always saved in the Messages tab above.
To also receive them by email, set up free EmailJS below:

Quick Setup Guide

  1. Go to emailjs.com and create a free account
  2. Add an Email Service (Gmail, Outlook, etc.) → copy the Service ID
  3. Create an Email Template using these variables: {{from_name}} {{from_email}} {{subject}} {{message}}
  4. Copy the Template ID and your Public Key (Account → API Keys)
  5. Paste them below and click Save Email Settings
